# Random terrain generation for Blender

Blender Addon for random terrain generation through noise maps.

## Installation

1. Download the addon .zip file from the releases
2. Open Blender and go to **Edit > Preferences**
3. Navigate to the **Add-ons** tab
4. Click **Install...** in the top-right corner
5. Select the downloaded addon .zip file
6. Enable the addon by checking the box next to its name

## Usage

After installation:
- **View > (enable) Sidebar** or press **N**
- **Noise** panel should appear on the right side
- **Delete all** objects and **Create grid**
- Play with the sliders to generate unique terrain
